## Installing in Linux/WSL

```bash
git clone
mkdir debug
cd tea_avl/debug
cmake ../src/ -DCAME_INSTALL_PREFIX=../tmp -DCMAKE_BUILD-TYPE=Debug
make
```

## Test
In the debug folder, run
```bash
./test-tree-avl
```
